# budget-tracker
![MIT license badge](https://img.shields.io/badge/license-MIT-brightgreen)

## Description
This app is a simple budget tracker. This app is a PWA and works with or without internet connection. The app allows you to keep track of your budget with the options to add funds and subtract funds.
